## Mac terminal SHortcuts

### Tab
```
Command + T ( new tab )
Control + Command + T ( with current  command )
Command-Grave Accent (`) ( toggle between tab )
Command-W ( close tab )
Shift-Command-T ( show or hide tab bar )
Shift-Command-Backslash (\) - show all tabs or exit tab overview )
Control-Tab ( next tab )
Control-Shift-Tab ( previous tab )
Command-W ( close tab )
Option-Command-W ( close other tabs )
Option-Shift-Command-W ( close all)

```

### Window
```
Command + N ( new window )
Option + Command + N ( with current  command )
Command-Grave Accent (`) ( toggle between windows )
Shift + Command-W ( close window )

```
